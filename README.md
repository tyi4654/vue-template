# vue-ant-pms

## 安装项目依赖

```
npm install
```

### 本地开发环境运行

```
npm run serve
```

### 测试开发环境运行

```
npm run serve:test
```

### 测试生产环境打包

```
npm run serve:debug
```

### 正式生产环境打包

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
